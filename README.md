# Cypress Todos

A rudimentary set of tests to be run against an example application

## Installation

Use the package manager npm to install.

```bash
npm install -g yarn
cd cypress-realworld-testing-todomvc
yarn install
```

## Start The App Under Test

```bash
yarn start
```

## Open The Cypress Dash

```bash
yarn cypress:open
```
